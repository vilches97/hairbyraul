# Hair by Raul

Sitio web para la peluquería Hair by Raul en Martos, especializada en novias, rubios bonitos, balayage y mechas.

## Características

- Diseño responsive para todos los dispositivos
- Sistema de reservas online
- Galería de trabajos
- Información de contacto y ubicación

## Tecnologías

- React 18 con TypeScript
- Vite como bundler
- Tailwind CSS para estilos
- Radix UI para componentes accesibles

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
